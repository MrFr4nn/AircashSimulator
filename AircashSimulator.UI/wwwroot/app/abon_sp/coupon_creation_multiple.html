Multiple coupon creation

The method used to let the sales partner request a creation of multiple new Aircash Abons at once. Aircash Abons are created using the CreateCouponMultiple method that creates the coupons and immediately activates them.

Environment Method URL
Test POST https://staging-a-bon.aircash.eu/rest/api/CashRegister/CreateMultipleCoupons
Production POST https://a-bon.aircash.eu/rest/api/CashRegister/CreateMultipleCoupons

Parameter Type Description Mandatory Part of the sequence
PartnerId string Unique partner identifier (per currency) provided by Aircash YES YES
PointOfSaleId string Identifier of the point of sale YES YES
ISOCurrencySymbol string Currency code by ISO 4217 (e.g. for EUR) YES YES
ContentType string Coupon format – „pdf“, „txt“ or „xml“. If the parameter is invalid or missing, default partner settings are used NO YES
ContentWidth int Width of the coupon (in mm for pdf, or character number for txt) Minimal value is 20. If the parameter is invalid or missing, default partner settings are used NO YES
Denominations List<object> List of objects containing info specific for each coupon to be created YES YES
Value decimal Coupon value, contained in denominations list YES YES
PartnerTransactionId string Unique identifier of the transaction in partner's system, contained in denominations list YES YES

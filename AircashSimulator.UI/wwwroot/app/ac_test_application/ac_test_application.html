Aircash test application


Follow the next instruction steps in order to successfully download, install and activate the Aircash test application. Note that the test application is for Android platform only.

Instructions:

  1. Send an email to info@aircash.eu with a list of emails of the test users to whom the invitation should be sent.
  2. Once the invitation is received, you need to download and install an App Tester application - an application that is used for distribution of Aircash test application represented with Illustration1.

    Illustration1: Download invitation

  3. Within the App Tester application you will have an option for installing the Aircash test app in which you will be notified that in order to continue with the installation you need to approve the app from an unknown source. Agree with that and resume installation.
  4. After successfully installing the Aircash test application, register with your email and mobile number (Illustration2). You can skip the video verification process (Illustration3). In case that you have not received infoSMS with registration code, contact our support on . If you already have existing account, login by clicking on button “I am already a user” (Illustration4) and choose login method (Illustration5).

  5. After successful registration, contact our support on info@aircash.eu to enable your testing environment and to top up testing accounts with funds.

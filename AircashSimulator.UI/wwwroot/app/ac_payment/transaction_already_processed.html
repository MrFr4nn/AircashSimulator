Transaction Already Processed

Option A

When receiving a CreateAndConfirmPayment request with the same TransactionID (this is Aircash transaction id) you can return the same successful response.

Response example:

Response HTTP 200 OK
Option B

When receiving a CreateAndConfirmPayment request with the same TransactionID (this is Aircash transaction id) you can return the error code for the already processed transaction.
You need to provide us with error code you will use for the already processed transaction. We will configure error code: e.g. error code 501 means transaction already processed, mark the transaction as a succesfull.

Response example:

Response HTTP 200 OK

FAQ


1. Q:  Is it possible to pay/make online deposits on sites that use multiple currencies?
A:  Yes it is. From a technical perspective, for each currency there needs to be a unique PartnerId parameter within the requests.
2. Q:  At what time is the payment visible?
A:  It is processed instantly.
3. Q:  What happens if the user does not have installed the Aircash app and wants to make the payment?
A:  After completing the purchase flow, if the user does not have Aircash app, he will be redirected to one of the stores to download the app.
4. Q:  Is it possible to make partial refunds?
A:  Yes it is possible. Refund method can be called multiple times as long as the sum of previous refund amounts is less or equal to the amount of the original transaction.
5. Q:  Is the QR code going to be visible within the mobile browser for making the payments/deposits?
A:  No. Mobile version has to implement a button that will have within it url that is behind the QR code and which is sent from us.

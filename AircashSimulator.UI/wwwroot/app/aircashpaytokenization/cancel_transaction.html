CancelTransaction


The Aircash system will not allow partners to cancel one transaction twice. In case of a double cancellation request, the Aircash system will process only the first request. For every other request, Aircash system will not process the cancellation request but it will return the same response as in the first cancellation request.

The cancellation method is idempotent, and the partner system can perform multiple requests until the partner system receives a successful response.

URL: /api/AircashPay/CancelTransaction

Parameter Type Description Mandatory
PartnerID string PartnerID YES
PartnerTransactionID string ID that partner sends in the Authorization method. YES
Signature string Transaction signature YES

Parameter Type Description Mandatory
PartnerID string PartnerID YES

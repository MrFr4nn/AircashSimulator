Reconciliation API

Method used to get transaction in selected period. Partner choses start and end dates and gets transactions that happend in selected period.

Partner can get transaction for only 14 days, if he tries to get transaction for longer period he will get error

Environment Method URL
Test GET https://dev-aircash.eu/api/
Production GET https://aircash.eu/api/

Parameters Type Description Mandatory
PartnerId String Partner identifier YES
StartDateUTC Date Date from witch partner wants transactions YES
EndDateUTC Date Date too witch partner wants transactions YES
Parameters Type Description
AircashTransactionId String Transaction id in Aircash system
partnerTransactionId String Transaction id in Partner system
DateTimeUTC DateTime Date and time when transaction was made
Amount decimal Transaction amount
currencyId int ISO-4217 currency code

*NOTE: Parameters in the table are included in each response, but you need to support new parameters as well, although you may not use them. If Aircash adds a new parameter, your system needs to accept the response with additional parameter(s).

HTTP 200
Http response Description Error Code Message Solution
400 An invalid request was made. The response will contain an error code and message. 1 Invalid partner id Contact Aircash support
2 Selected period too long
500 An unknown error occured - Unknown error has occurred Contact Aircash support
HTTP 400
